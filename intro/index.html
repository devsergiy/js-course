        var myArr = [];// (1) <- добавлена
        // (2) Не добавлена, строка убирается
        console.log('test')(myArr || []).forEach(function(i) {});

Обзор возможностей сlient-side JavaScript

JavaScript позволяет:

  • Работать с DOM (создание/удаление элементов и изменение их свойств);
  • Перехватывать действия пользователя(клики мышки, движение курсора, набор текста и др.);
  • Показывать сообщения(alerts);
  • Устанавливать cookies;
  • Посылать http запросы;

JavaScript не может:

  • Не имеет прямого доступа к памяти и к файловой системе;
  • Нет доступа к соседним вкладкам, если они не были открыты нами;
  • Ограниченные возможности кросс-доменного доступа;
