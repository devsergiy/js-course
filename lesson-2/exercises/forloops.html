Генератор четных - нечетных чисел

Создайте цикл, который будет случайным образом генерировать числа от 0 до 20. Для каждого случая будет проверятся текущее число (четное или нечетное), и выводится результат на экран (например, 2 - четное). 



for (var i = 0; i <= 20; i++) {
    if (i % 2 === 0) {
        console.log(i + ' is even');
    } else {
        console.log(i + ' is odd');
    }
}

Оценка результатов

Проверьте результаты функции assignGrade из условных упражнений для каждого значения от 60 до 100. То есть создайте цикл for от 60 до 100 и вызывайте в нем вашу функцию, например assignGrade(61) 

function assignGrade(score) {
    if (score > 90) {
        return '5';
    } else if (score > 80) {
        return '4';
    } else if (score > 70) {
        return '3';
    } else if (score > 65) {
        return '2';
    } else {
        return '1';
    }
}

for (var i = 80; i <= 100; i++) {
  console.log('For ' + i + ', you got a ' + assignGrade(i));
}
