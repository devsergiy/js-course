Калькулятор возраста

  • Написать функцию calculateAge которая:
    • принимает 2 аргумента: год рождения и текущий год.
    • Считает два возможных значения возраста.
    • И выводит их в следующем виде: "Ваш возраст NN или NN"
  • Вызвать функцию с двумя разными наборами данных
  • Bonus: Выяснить как можно получить текущий год не передавая его. 


function calculateAge(birthYear, currentYear) {
    var age = currentYear - birthYear;
    console.log('Ваш возраст ' + age + ' или ' + (age - 1));
}

calculateAge(1984, 2012);
calculateAge(1988, 2012);
calculateAge(1982, 2012);
